# PickCircle

友達だけで遊ぶ、換金不可ポイント制のスポーツ予想リーグMVPです。

## ファイル

- `index.html`  
  単体で動くデモアプリです。ブラウザで開くと、リーグ、試合、予想、結果入力、ランキングを試せます。

## 機能

- 友達限定リーグ風ダッシュボード
- 試合作成
- 倍率風ポイントの手入力
- 予想入力
- 結果確定
- 自動採点
- ランキング
- 予想履歴
- メンバー追加
- ブラウザのローカル保存

## 注意

このMVPは、現金、景品、換金可能ポイント、参加費を原資にした賞金配分を扱いません。ポイントはリーグ内の順位表示だけに使います。
