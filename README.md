# Yoso League

仲間内のスポーツ予想リーグを、ポイント制で管理する静的Webアプリです。

## 使い方

`index.html` をブラウザで開くとそのまま動きます。データは端末の `localStorage` に保存されます。

## プリセット

- 順位予想: WBCのような順位ポイント × オッズ
- ドラフト: 選抜高校野球のようなチーム指名制
- ファイトカード: ボクシングの勝敗・決着方法・ラウンド帯予想
- W杯3フェーズ: グループリーグ、決勝トーナメント、複勝オッズ、決勝スコアを合算

## App Store向けの設計方針

このアプリはリアルマネーのベット、送金、決済、ブックメーカー誘導を行いません。  
用途は、プライベートグループ内のスポーツ予想をポイントで記録することに限定しています。

将来iOSアプリ化する場合も、審査通過を優先するなら以下を維持してください。

- アプリ内で掛け金や支払いを徴収しない
- 決済・送金リンクを組み込まない
- 外部ブックメーカーへの登録・入金誘導をしない
- 表現は「予想」「ポイント」「リーグ」「ランキング」に寄せる
